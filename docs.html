Primeiros Passos

Para começar, você precisa criar uma conta e obter sua chave de API. Siga os passos abaixo:

  1. Crie sua conta na página de registro.
  2. Navegue até o seu dashboard e gere uma chave de API.
  3. Use a chave de API no header de suas requisições.
curl -H "Authorization: Bearer SUA_CHAVE" https://api.automa.io/v1/...

Referência da API

Nossa API é organizada em torno dos princípios REST. Nossos endpoints aceitam requisições JSON e retornam respostas JSON, utilizando os códigos de status HTTP padrão.

Endpoint: Criar Automação

POST /automations

Este endpoint cria uma nova automação baseada nos dados enviados. (Este é o endpoint que usamos no nosso JSON Server!)
